    Running from Responsibilities

    Running from Responsibilities is an endless, runner-style arcade game (think subway surfers, but with homework and chores as the obstacles) that I created with a few of my friends for Hack the North 2021 (a hackathon). We used Tensorflow and Pygame to create an app that tracked real-life user movements to play the game (resulting in the ultimate game to play while taking a break from schoolwork!), and I worked on obstacle generation and movement using Pygame.

    Sept 2021

    This project was my first introduction to game development, and opened my eyes to the applications of Python. I had never realized that the programming languages I had previously only used competitively could create video games. This project also gave me a glimpse of all the collaboration needed to put a project together. As each member of the group was working on a specific aspect or function, we had to link all the different aspects together, and could not test the project without having every aspect present. This presented some challenges while debugging, as there were so many functions working together that it was difficult to pinpoint where exactly we went wrong.

    Notebooks Plus

    Notebooks Plus is a web application designed for desktops that allows for users to take time-stamped notes alongside recorded lectures or videos. Amidst the pandemic, we felt this app would help fellow students who came to rely on videos for online school. I created this project with a group of friends for Hack the North 2020 (a hackathon). The web app was created using HTML, CSS, JavaScript, and TypeScript, although I worked exclusively on the front-end of the project. I was responsible for creating and implementing our design for the various pages of our app, including the home page, folder view, and notebook view.
